❓ Frequently Asked Questions

Rozgar Sathi bilkul free hai?
Haan, Rozgar Sathi 100% free hai. Koi subscription ya payment nahi. Ads se platform maintain hota hai.
Ek login se teeno apps mein kaise kaam karta hai?
Teeno apps (Samachar Sathi, Pariksha Sathi, Rozgar Sathi) same Firebase project use karte hain. Ek baar Google se login karo, teeno mein automatically logged in ho jaate ho.
Mera resume data safe hai?
Haan. Resume data aapke browser ke localStorage mein save hota hai. Login karne par Firebase Firestore mein sync hota hai jo encrypted hai. Hum aapka data kabhi share nahi karte.
Naya sarkari vacancy add karwana hai, kaise karein?
Upar contact form mein "Naukri Suggestion" select karke vacancy ki details bhejein, ya GitHub Issues par submit karein. Hum jald add kar denge.
Apna account aur data delete karna hai?
Contact form mein "Data Deletion Request" select karke apna registered email bhejein. 7 working days mein aapka data delete kar diya jayega.
Apply link official website par le jaata hai?
Haan, bilkul. Sabhi apply links official government portals (ssc.gov.in, rrbapply.gov.in, ibps.in, etc.) par redirect karte hain. Hum koi third-party form nahi use karte.
